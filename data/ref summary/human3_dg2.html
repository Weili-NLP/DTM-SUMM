[1] Sentiment Analysis or Opinion Mining is about detecting people’s opinions, attitudes and emotions toward an product or entity. [2] Sentiment Analysis is on three levels document level, phrase level, and aspect level Sentiment Analysis. [3] Aspect-level Sentiment Analysis aims to classify the sentiment polarity is positive, negative or neutral with respect to the specific aspects (price) of products or entities (food). [4] Classification approaches such as Naive Bayes (NB) and Support Vector Machine (SVM) using machine learning to identify sentiment polarity exploit sentiment lexicon resources like sentiWordNet, Bing Liu’s opinion lexicon, IMDB, MPQA and Sentiment 140 lexicon. [5] Attributes-based sentiment analysis is to analyze sentiment based on each attribute of a product by mining sentiment polarity of product features. [6] Typical Topic models for sentiment analysis are Joint sentiment topic model (JST), Aspect and Sentiment Unification model (ASUM), Topic Sentiment Mixture (TSM) for unsupervised joint sentiment topic detection. [7] Contract in LDA, these models assume that topics are generated dependent on sentiment distributions and then words are generated conditioned on sentiment-topic pairs. [8] Domain-specific ontology tree like Sentiment Ontology Tree SOT are used to feature-level aspects of a product. [9] Sentiment Ontology Tree is a semantic representation of the opinionated attributes of a product. [10] The performance for sentiment classification is evaluated by metrics like precision, recall and F-measure.
