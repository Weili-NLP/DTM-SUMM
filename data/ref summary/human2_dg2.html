[1] Sentiment analysis or opinion mining involves examine opinions about the products and aspects (price, food) in reviews or tweets. [2] Sentiment analysis has document level, word level, phrase level and attribute level. [3] Sentiment Classification techniques can be roughly divided into machine learning approach, lexicon based approach and hybrid approach. [4] Opinion words are indentified in many sentiment classification tasks using sentiment lexicons: Bing Liu’s opinion lexicon, IMDB, MPQA, SentiWordNet, NRC Emotion Association lexicon, Sentiment 140 lexicon and NRC Hashtag lexicon. [5] The aspect term polarity classification is to determine whether the sentiment polarity of each aspect is positive, negative or neutral. [6] A taxonomy (knowledge) based approach to extracting feature-specific sentiments to determine the overall review polarity relies on Sentiment Ontology Tree SOT in a specific domain. [7] Sentiment Ontology Tree SOT represents the relationships between a product’s attributes or features. [8] Unsupervised learning and upervised classification (NB, SVM) methods are used to determine the sentiment polarity of each aspect category. [9] Data sets are movie reviews, in camera domain and multi-domain sentiment data set from IMDB and amazon.com. [10] The performance for sentiment classification is evaluated by metrics like precision, recall and F-measure.
